⚙️ Configurar Supabase

Para activar la plataforma necesitas una cuenta gratuita en supabase.com.
Crea un proyecto, luego una tabla videos y un bucket de Storage llamado videos. Pega tus credenciales aquí — se guardan solo en tu navegador.

📋 SQL para crear la tabla (haz clic) 
create table videos (
  id uuid default gen_random_uuid() primary key,
  title text not null,
  description text,
  video_url text not null,
  thumb_url text,
  created_at timestamptz default now()
);

-- Habilitar acceso público (RLS off para demo)
alter table videos enable row level security;
create policy "public read" on videos for select using (true);
create policy "public insert" on videos for insert with check (true);
